# Data Cleaning Pipeline Project

This project demonstrates basic data cleaning using Python and Pandas.

## Features
- Reads messy CSV dataset
- Handles missing values
- Removes duplicates
- Cleans invalid data
- Outputs cleaned CSV file

## Technologies Used
- Python
- Pandas
- VS Code
- Git & GitHub

## Files
- `raw_data.csv` → Original dataset
- `cleaned_data.csv` → Cleaned output dataset
- `main.py` → Main cleaning script
